## Instruções de execução

Para correr o projeto é necessário ir até à diretoria onde está o ficheiro **pom.xml**.

Primeiro iniciar o **servidor** com o seguinte comando:

```
mvn exec:java
```

De seguida executar o **cliente** com o comando:
```
mvn exec:java
```
